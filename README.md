<h2> Installing and running the project 🎲</h2>

```bash
# Clone este repositório
$ git clone https://github.com/GusttavoCastilho/Hekto

# Acesse a pasta do projeto no seu terminal/cmd
$ cd Hekto

# Instale as dependências
$ yarn install

# Inicie o Servidor
$ yarn start

```
